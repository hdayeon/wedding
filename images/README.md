# 사진 넣는 곳

아래 이름으로 이 폴더에 넣으면 자동으로 표시됩니다.
파일이 없으면 회색 자리표시 이미지가 대신 나옵니다.

| 파일명 | 용도 | 권장 크기 |
|---|---|---|
| `main.jpg` | 표지 사진 | 세로 3:4, 긴 변 1600px |
| `01.jpg` ~ `09.jpg` | 갤러리 | 정사각 1:1, 1200×1200 |
| `og.jpg` | 링크를 **붙여넣었을** 때 뜨는 미리보기 | 가로 1200×630 |
| `share.jpg` | **공유 버튼**으로 보낸 카톡 메시지의 큰 이미지 | 세로 640×852 (3:4) |

`og.jpg`와 `share.jpg`는 쓰임이 다릅니다. 자세한 건 [최상위 README](../README.md#카카오톡-공유-두-가지가-다릅니다)를 보세요.
`share.jpg`가 세로 3:4가 아니면 카톡 말풍선에서 이미지가 작게 나옵니다.

## 용량을 꼭 줄여주세요

원본 사진을 그대로 올리면 한 장에 5~10MB라 모바일에서 로딩이 오래 걸립니다.
갤러리 9장이면 수십 MB가 되어 하객이 기다리다 나갑니다.
**한 장당 300KB 이하**를 목표로 압축하세요.

macOS에서 한 번에 줄이는 방법 (ImageMagick 필요: `brew install imagemagick`):

```sh
# 갤러리용 정사각 1200px, 품질 82
magick mogrify -resize 1200x1200^ -gravity center -extent 1200x1200 -quality 82 0*.jpg

# 표지용 긴 변 1600px
magick mogrify -resize 1600x1600\> -quality 82 main.jpg
```

사진 장수를 바꾸려면 `index.html`의 `CONFIG.photos.gallery` 배열을 수정하세요.

## 이름과 얼굴이 공개됩니다

이 저장소는 GitHub Pages 무료 플랜 때문에 공개 상태입니다.
올린 사진은 누구나 볼 수 있고, 커밋 기록에도 남습니다.
나중에 지워도 커밋 히스토리에는 남으니, 올리기 전에 한 번 더 확인하세요.
